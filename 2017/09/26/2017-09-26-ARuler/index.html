Ruler

A simple AR ruler app can measure length & area

Requirements

  • Swift 4.0
  • iOS 11.0
  • Xcode 9.0

Preview

  1. Mesure Length

youtube : https://youtu.be/2wKLUqPizfQ

  1. Mesure Area

Install

  • AppStore (recommend)

    ARuler https://itunes.apple.com/us/app/id1287068680

  • Manually compile
    1. git clone https://github.com/TBXark/Ruler
    2. pod install
    3. Compile project with Xcode
